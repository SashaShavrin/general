ShopWear
Implementation of a multi-page online clothing store. The main idea of the project was to create a counter of goods. The counter correctly counts the number of selected products and summarizes them in the basket. The main implementation of the counter was reading click and mouseover events.
The counter saves the value when switching from page to page, as well as when closing and restarting the computer. Saving the counter - localStorage.setItem, getting the counter on another page - localStorage.getItem
